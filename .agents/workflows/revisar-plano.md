# Workflow: Revisar Plano (PLANO-REVISAO.md)

Leia a raíz de `PLANO-REVISAO.md`.

As sub-tarefas estão divididas em REVISÕES:
- REVISÃO 1 — Numerologia Core (backend)
- REVISÃO 2 — Banco de Dados
- REVISÃO 3 — Produtos (lógica específica)
- REVISÃO 4 — AI Prompts
- REVISÃO 5 — Frontend
- REVISÃO 6 — Skills `GEMINI.md` e `CLAUDE.md`

## O que fazer
1. Liste os arquivos necessários para cada passo.
2. Certifique-se de que os passos sejam feitos numéricos (Refatorando primeiro a Númerologia, para depois atualizar a DB, a lógica, Front End e AI Prompts).
3. Após criar ou atualizar scripts, edite o "Checklist final" e "Ordem de Execução" atualizando e documentando o que resta para concluir.

**IMPORTANTE:**
Sempre que uma mudança em backend/núcleo for concluída, teste as assinaturas da DB ou interfaces se estiverem coesas com O TYPESCRIPT estabelecido. Tudo que você fizer reflete o projeto inteiro.
